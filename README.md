# 🦈 鯊逼多益 - TOEIC 練習 App

一個 TOEIC 練習 Web App，使用 Next.js + TypeScript + Tailwind CSS 開發。

## 功能

- 📝 三種測驗模式：單字、文法、模擬考
- ⏱️ 橫條計時器（藍→橙→紅）
- 📖 結果頁面含題目解析
- ⚠️ 追蹤長思考題目（>30秒）
- 📚 我的單字本功能
- ❤️ 長按單字加入生詞本（自動查詢 Dictionary API）

## 技術

- Next.js 15
- TypeScript
- Tailwind CSS
- Shadcn UI
- localStorage 儲存

## 開始使用

```bash
npm install
npm run dev
```

開啟 http://localhost:3000

## 部署

支援 Docker 部署到 Unraid。
